Azkar API 📿

A simple and structured API that provides a collection of Islamic Azkar (remembrances), including morning and evening supplications, sleep Azkar, Ruqyah (Islamic healing), and more.

📌 Overview

This project is designed to serve as a backend data source for frontend applications. It allows developers to easily fetch and display Azkar content in their apps.

📚 Content Included
Morning Azkar ☀️
Evening Azkar 🌙
Sleep Azkar 😴
Ruqyah Shariah 🛡️
General Azkar 📿
⚙️ Use Cases
Islamic mobile/web applications
Reminder apps
Educational platforms
🛠️ Technologies
JSON / REST API
🚀 How to Use

You can fetch the data and integrate it into your frontend project easily.

Example:

fetch('/api/azkar')
❤️ الهدف

Providing easy access to authentic Azkar for developers and users to benefit from daily remembrance.
